# House Rent Management System

A full-stack MERN application for house rental management.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Registration & Login
- Browse House Listings
- View Property Details
- Add Property
- Book Houses
- Manage Bookings
